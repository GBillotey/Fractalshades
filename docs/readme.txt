To build the docs you will need to install sphinx, sphinx_rtd_theme etc...


> python3 -m pip install --user sphinx
> python3 -m pip install --user sphinx_rtd_theme
> python3 -m pip install --user sphinx-gallery


Then (under Linux) from a command prompt in this directory :
> make html 


https://sphinx-gallery.github.io/stable/getting_started.html
