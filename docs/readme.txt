To build the docs you will need to install sphinx, sphinx_rtd_theme etc...

> python3 -m pip install  sphinx
> python3 -m pip install  sphinx_rtd_theme
> python3 -m pip install  sphinx-gallery
> python3 -m pip install numpydoc
> python3 -m pip install groundwork-sphinx-theme
> python3 -m pip install sphinxcontrib-bibtex
> python3 -m pip install sphinxcontrib-youtube

> python3 -m pip install scipy
> python3 -m pip install av

> sudo apt-get install dvipng
> sudo apt-get install texlive
> sudo apt-get install texlive-latex-extra

Then (under Linux) from a command prompt in this directory :
> make html
